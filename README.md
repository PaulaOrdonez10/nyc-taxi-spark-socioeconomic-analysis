# NYC Taxi Analytics & Socioeconomic Insights

## Overview

This project analyses over 12 million New York City taxi trips using Apache Spark and Databricks. The objective was to process large-scale transportation data, perform geospatial analysis, and enrich taxi trips with socioeconomic indicators such as poverty and unemployment rates.

Developed as part of the Master's in Data Science, Big Data & Business Analytics at Universidad Complutense de Madrid.

---

## Technologies Used

- Apache Spark (PySpark)
- Databricks
- Azure Storage
- Python
- Pandas
- GeoPandas
- Folium

---

## Project Objectives

- Clean and validate large-scale taxi trip data.
- Perform feature engineering on transportation records.
- Identify and analyse geospatial outliers.
- Enrich trips with district-level socioeconomic indicators.
- Explore relationships between tipping behaviour, poverty and unemployment rates.

---

## Key Results

- Processed over 12 million taxi trip records.
- Reduced dataset to approximately 11.5 million valid trips after cleaning.
- Enriched more than 1 million trips with socioeconomic information.
- Performed correlation analysis between tipping behaviour and socioeconomic variables.
